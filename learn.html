Learning Notes

Read the following notes carefully before starting the quiz.

HTML

HTML (HyperText Markup Language) is used to create the structure of web pages. It contains elements such as headings, paragraphs, images, links, tables, and forms.

CSS

CSS (Cascading Style Sheets) is used to style web pages. It controls colors, fonts, spacing, layouts, animations, and responsive design.

JavaScript

JavaScript adds interactivity to websites. It can validate forms, create animations, handle events, and communicate with servers.

Python

Python is a simple and powerful programming language. It is widely used in web development, artificial intelligence, machine learning, automation, and data science.

Flask

Flask is a lightweight Python web framework. It helps developers build websites and web applications using Python.

SQLite

SQLite is a lightweight database. It stores user information, quiz scores, and application data without requiring a separate server.
